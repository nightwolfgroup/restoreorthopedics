Special Notes on SEO

Any page on your website can be password protected. This may be useful in instances where you want to place content behind a pay-wall.

Password protecting a page will also apply that protection to any child pages. For example, if you were to password protect the Home Page, all pages of the website will require the password to access them as all pages on the site are technically children of the home page.

Click Pages in the Navigation Menu then

open the editor for the page you would like to add password protection to.

Open the "Page Settings Tab"

Click "Set Page Privacy"

Select "Private, accessible with the following password"

Set the Page Password

After password protection has been set, the password will be required, even for logged-in users
