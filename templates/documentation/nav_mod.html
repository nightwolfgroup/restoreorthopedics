Modify Navigation Header & Footer

Updating Links in the Main Navigation Bar

If you are not seeing a link appear when you are linking to an internal page, click the "Edit This Page" button and navigate to the "SEO/MENU" tab. Ensure "shown in menus" is checked.

Click Settings in the Navigation Menu then

click Main Menu in the Settings Menu

Note the order of these items. They correspond with the order they will appear in the navigation bar at the top of the website.

Linking to an Internal Page

If you are creating a link to an internal page, simply click "Choose Page" or "Choose Another Page"

Select the page you would like the link to navigate to when clicked

Linking to an External Page

If you are creating a link to an external page, leave the selection "Link to an internal page" blank

Place the URL you would like the link to navigate to when clicked in the field "Link to a custom URL"

Allowing a Sub-Menu

It is important to note that sub-menus are automatically generated based on the hierarchy of the website. For example, I cannot place a link to a blog post under another section as blog posts are only children pages of the blog. If a page has children and you select the checkbox in this menu to enable the sub-menu, all child pages one-level below the parent page (with the show in menus button checked) will display in the navigation bar when the link is hovered.
