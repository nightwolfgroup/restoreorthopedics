Managing Images & Documents

Image Management

If an image is ever accidentally deleted, contact me and I will pull it from the server for you. A backup copy of all images are stored on the server for situations like this. Deleted images will be saved for six months.

Click Images in the Navigation Menu

Image Management Panel

Uploading an Image

Click "Add an Image

You can either click the button, or drag the image you wish to add into the square on the page

Editing Image Attributes

  • Title: The title is used as the "alt" attribute on the frontend of the site. If a user is blind or using a screen reader, this is what the screen reader will read to the user.
  • File: This is the name of the file as the server has named it. If you click this link it will allow you to download it to your computer. To change the image, click "Choose File" and upload a replacement.
  • Tags: Tags are used internally for locating image files. As your image library starts to grow, you will find tags helpful in locating older images. I have tagged the images on your site with the type (i.e. 'icon') and the page it is on (i.e. 'icon').
  • Focal Point: There may be occasions where you upload an image, however, upon publishing of the page the image is not positioned appropriately (top/bottom are cut off). Adding a focal point tells your website the most important part of the image, this way it will not cut that part off when the page loads.
